# Need for Speed

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Need for Speed (Exercism)](https://exercism.org/tracks/java/exercises/need-for-speed)

The goal of the exercise is to practice working with:

- Constructors
- Classes and objects
- Instance fields
- Static factory methods
- Object interaction
- Basic arithmetic and conditional logic

## What I Learned

While solving this exercise, I practiced:

- Initializing objects using constructors
- Tracking and updating an object's state
- Creating objects through a static factory method
- Using one object's data inside another class
- Calculating whether an object can complete a task based on its current state
- Working with integer division and the modulo (`%`) operator
