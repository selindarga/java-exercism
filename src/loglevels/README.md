# Log Levels

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Log Levels (Exercism)](https://exercism.org/tracks/java/exercises/log-levels)

The goal of the exercise is to practice working with:

- String manipulation
- String parsing
- Static methods
- Common `String` methods

## What I Learned

While solving this exercise, I practiced:

- Extracting parts of a string using `substring()`
- Finding character positions with `indexOf()`
- Removing leading and trailing whitespace using `trim()`
- Converting text to lowercase with `toLowerCase()`
- Reusing helper methods to avoid code duplication
