# Captain's Log

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Captain's Log (Exercism)](https://exercism.org/tracks/java/exercises/captains-log)

The goal of the exercise is to practice working with:

- The `Random` class
- Constructor injection
- Arrays
- Generating random values
- Constants

## What I Learned

While solving this exercise, I practiced:

- Using a provided `Random` instance instead of creating new ones
- Generating random elements from an array
- Generating random numbers within a specific range
- Writing simple methods that produce random values
