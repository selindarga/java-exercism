# Cars Assemble

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Cars Assemble (Exercism)](https://exercism.org/tracks/java/exercises/cars-assemble)

The goal of the exercise is to practice working with:

- Conditional statements (`if` / `else if` / `else`)
- Numeric calculations
- Type casting
- Method reuse

## What I Learned

While solving this exercise, I practiced:

- Applying different production success rates based on speed
- Reusing methods instead of duplicating logic
- Converting `double` values to `int` using explicit casting
- Improving code readability by using constants instead of magic numbers
