# Logs, Logs, Logs!

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Logs, Logs, Logs! (Exercism)](https://exercism.org/tracks/java/exercises/logs-logs-logs)

The goal of the exercise is to practice working with:

- Enums
- Enum constructors and fields
- `switch` statements
- String parsing
- Object-oriented design

## What I Learned

While solving this exercise, I practiced:

- Defining enums with custom fields and constructors
- Associating additional data with enum constants
- Using `switch` statements to map string values to enum constants
- Parsing structured log messages using string methods
- Reusing methods to avoid code duplication and improve readability
