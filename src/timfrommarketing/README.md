# Tim from Marketing

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Tim from Marketing (Exercism)](https://exercism.org/tracks/java/exercises/tim-from-marketing)

The goal of the exercise is to practice working with:

- Conditional statements (`if` / `else`)
- The ternary operator
- Handling `null` values
- String manipulation
- Building formatted output

## What I Learned

While solving this exercise, I practiced:

- Checking for `null` values before using object methods
- Using the ternary operator to simplify conditional logic
- Formatting strings based on different input conditions
- Converting text to uppercase using `toUpperCase()`
- Reducing code duplication by extracting common logic into a variable
