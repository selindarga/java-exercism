# Booking Up For Beauty

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** https://exercism.org/tracks/java/exercises/booking-up-for-beauty

The goal of the exercise is to practice working with:

- The `java.time` API
- `LocalDate` and `LocalDateTime`
- `DateTimeFormatter`
- Parsing and formatting dates and times
- Comparing dates and times

## What I Learned

While solving this exercise, I practiced:

- Parsing date and time strings into `LocalDateTime`
- Formatting dates and times using `DateTimeFormatter`
- Comparing dates with the current date and time using `isBefore()`
- Determining whether an appointment falls within a specific time range
- Retrieving individual date and time components such as the year, hour, and day of the week
- Working with `LocalDate` to generate dates for the current year
- Formatting output using different patterns and `Locale.ENGLISH`
