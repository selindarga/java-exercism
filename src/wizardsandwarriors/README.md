# Wizards and Warriors

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Wizards and Warriors (Exercism)](https://exercism.org/tracks/java/exercises/wizards-and-warriors)

The goal of the exercise is to practice working with:

- Inheritance
- Method overriding
- Polymorphism
- Instance variables
- Object state

## What I Learned

While solving this exercise, I practiced:

- Creating subclasses using inheritance
- Overriding methods to provide specialized behavior
- Using polymorphism to call the appropriate method implementation at runtime
- Managing object state with instance variables
- Implementing different behaviors based on an object's current state
