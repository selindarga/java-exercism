# Bird Watcher

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Bird Watcher (Exercism)](https://exercism.org/tracks/java/exercises/bird-watcher)

The goal of the exercise is to practice working with:

- Arrays
- Looping through arrays
- Conditional statements
- Basic array manipulation

## What I Learned

While solving this exercise, I practiced:

- Iterating through arrays using `for` and enhanced `for` loops
- Calculating totals from array elements
- Updating values stored in an array
- Using `Math.min()` to safely handle array boundaries
- Writing clean and readable methods for common array operations
