# Annalyn's Infiltration

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Annalyn's Infiltration (Exercism)](https://exercism.org/tracks/java/exercises/annalyns-infiltration)

The goal of the exercise is to practice working with:

- Boolean values
- Logical operators (`&&`, `||`, `!`)
- Returning boolean expressions directly

## What I Learned

While solving this exercise, I practiced:

- Simplifying boolean logic
- Replacing unnecessary `if` statements with direct boolean expressions
- Writing cleaner and more readable code
