# Jedlik's Toy Car

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Jedlik's Toy Car (Exercism)](https://exercism.org/tracks/java/exercises/jedliks-toy-car)

The goal of the exercise is to practice working with:

- Classes and objects
- Instance fields
- Static methods
- Constants (`static final`)
- Encapsulation (`private` fields)
- Basic conditional logic

## What I Learned

While solving this exercise, I practiced:

- Creating and instantiating Java objects
- Using instance fields to store object state
- Defining constants with `static final`
- Encapsulating data using `private` fields
- Updating an object's state through methods
- Returning formatted strings based on object state
