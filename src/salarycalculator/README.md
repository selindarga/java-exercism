# Salary Calculator

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Salary Calculator (Exercism)](https://exercism.org/tracks/java/exercises/salary-calculator)

The goal of the exercise is to practice working with:

- Conditional logic
- Methods
- Constants
- Arithmetic operations
- The ternary operator

## What I Learned

While solving this exercise, I practiced:

- Using the ternary operator to simplify conditional logic
- Breaking a calculation into small, reusable methods
- Defining constants with `static final`
- Performing calculations with different return types (`int` and `double`)
- Improving code readability by avoiding duplicated logic
