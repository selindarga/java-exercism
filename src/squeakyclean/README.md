# Squeaky Clean

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Squeaky Clean (Exercism)](https://exercism.org/tracks/java/exercises/squeaky-clean)

The goal of the exercise is to practice working with:

- Characters (`char`)
- Character utility methods
- StringBuilder
- String manipulation
- Switch statements

## What I Learned

While solving this exercise, I practiced:

- Using the `Character` class to inspect and transform characters
- Building strings efficiently with `StringBuilder`
- Converting kebab-case to camelCase
- Replacing leetspeak characters with letters
- Filtering unwanted characters
