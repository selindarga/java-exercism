# Calculator Conundrum

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Calculator Conundrum (Exercism)](https://exercism.org/tracks/java/exercises/calculator-conundrum)

The goal of the exercise is to practice working with:

- Exceptions
- Custom exceptions
- Exception handling (`try` / `catch`)
- Switch expressions
- String formatting

## What I Learned

While solving this exercise, I practiced:

- Throwing built-in exceptions such as `IllegalArgumentException`
- Catching exceptions with `try` / `catch`
- Wrapping exceptions by preserving the original cause
- Creating meaningful exception messages
- Using `switch` statements to handle different operations
