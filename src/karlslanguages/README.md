# Karls Languages

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Karl's Languages (Exercism)](https://exercism.org/tracks/java/exercises/karls-languages)

The goal of the exercise is to practice working with:

- Lists
- The `List` interface
- `ArrayList`
- Common collection methods

## What I Learned

While solving this exercise, I practiced:

- Creating and initializing an `ArrayList`
- Using the `List` interface instead of a concrete implementation
- Adding and removing elements from a list
- Retrieving elements and checking list contents
- Using built-in collection methods to write clean and concise code
