# Football Match Reports

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Football Match Reports (Exercism)](https://exercism.org/tracks/java/exercises/football-match-reports)

The goal of the exercise is to practice working with:

- `switch` statements
- Control flow
- Method return values
- Grouping multiple `case` labels

## What I Learned

While solving this exercise, I practiced:

- Using `switch` statements to simplify multiple conditional branches
- Grouping multiple `case` labels that share the same outcome
- Using the `default` case to handle unexpected input
- Returning values directly from `switch` cases
- Writing clean and readable control flow logic
