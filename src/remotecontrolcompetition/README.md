# Remote Control Competition

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Remote Control Competition (Exercism)](https://exercism.org/tracks/java/exercises/remote-control-competition)

The goal of the exercise is to practice working with:

- Interfaces
- Classes and objects
- Polymorphism
- Implementing multiple interfaces
- The `Comparable` interface
- Sorting collections
- Static methods

## What I Learned

While solving this exercise, I practiced:

- Defining and implementing Java interfaces
- Using polymorphism to write methods that work with different object types
- Implementing the `Comparable` interface to define a natural ordering
- Overriding the `compareTo()` method
- Sorting a list of objects using `Collections.sort()`
- Organizing related classes across multiple Java source files
