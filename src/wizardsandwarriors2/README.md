# Game Master

## About

This is a beginner-level Java exercise from Exercism.

**Exercise:** [Game Master (Exercism)](https://exercism.org/tracks/java/exercises/wizards-and-warriors-2)

The goal of the exercise is to practice working with:

- Method overloading
- Classes and objects
- Enums
- String concatenation
- Reusing methods to avoid code duplication

## What I Learned

While solving this exercise, I practiced:

- Implementing multiple overloaded methods with the same name but different parameter lists
- Working with classes and enums as method parameters
- Reusing existing methods to keep the code clean and maintainable
- Building descriptive strings using string concatenation
- Improving code readability by reducing duplicated logic
